# hapi-starter-sql

1) Clone repo
2) npm install
3) Create .env file
4) Create test and dev databases
5) Run knex migrate:latest 
6) Run knex migrate:latest --env test
7) Run knex seed:run
8) Run knex seed:run --env test
9) npm run test --should pass
10) npm run start
11) localhost:8080/documentation -- API docs
